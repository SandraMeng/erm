# 简介

员工管理系统，包括部门、岗位、员工管理模块，实现增删改查功能。

# 技术栈

vue + vuex + vue-cli + ES6/7 + axios + node + less + element-ui

# 项目运行

```	
# 克隆项目
(git bash) git clone https://github.com/SandraMeng/erm

# 安装依赖
cd erm-vue-client 
npm i 

cd erm-admin 
npm i 

# 启动服务
cd erm-admin
node server.js

cd erm-vue-client
npm run serve 或 yarn serve

# 访问项目
访问 http://localhost:8080
```
